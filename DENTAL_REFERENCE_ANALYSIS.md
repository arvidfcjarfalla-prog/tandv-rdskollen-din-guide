# Swedish Dental Reference Prices Analysis
## TLV Referenspriser från 1 januari 2026

**Document Date:** March 20, 2026
**Source File:** Referenspriser med dentalt material från 1 jan 2026.xlsx
**Total Treatments:** 200 unique treatment codes
**Validity:** January 1, 2026 onwards

---

## Executive Summary

The TLV reference price list contains **200 dental treatment procedures** organized into **9 main categories** (coded as 1xx through 9xx). Each treatment has:
- **Treatment code** (åtgärdsnummer) - unique identifier
- **Swedish description** - detailed treatment name
- **General dentistry price** (allmäntandvård) - price when performed by general dentist
- **Specialist price** (specialisttandvård) - price when performed by specialist (often higher)
- **Dental technical costs** - for lab-work related procedures
- **Material information** - for treatments with specific material specifications

---

## Key Insights

### Price Differences: General vs. Specialist

Specialist pricing is typically **higher** for the same procedure:

| Procedure | General | Specialist | Difference | % Increase |
|---|---|---|---|---|
| Basic examination (101) | 1,100 | 1,100 | 0 | 0% |
| Comprehensive exam (107) | 1,275 | 1,970 | +695 | +54% |
| Acuate exam (103) | 445 | 585 | +140 | +31% |
| Full mouth X-ray (123) | 1,020 | 1,595 | +575 | +56% |
| Panorama X-ray (124) | 635 | 1,130 | +495 | +78% |
| Tooth extraction (401) | 490 | 980 | +490 | +100% |
| Root canal 1 canal (501) | 2,105 | 2,605 | +500 | +24% |
| Crown (800) | 4,550 | 5,460 | +910 | +20% |

**Average specialist markup: 25-50% premium over general dentistry**

### Most Common Patient Treatments (By Code Frequency)

**1. Examinations (101-167)** - 27 treatments
- Entry point for most patient visits
- Price: 445-1,970 SEK (specialist can double cost for comprehensive exams)

**2. Fillings (701-708)** - 9 treatments
- Varying by number of surfaces and tooth type
- Price: 450-1,690 SEK
- Front/canine teeth slightly cheaper than molars/premolars

**3. Other Dental Care (800-891)** - 68 treatments
- Largest category (34% of all treatments)
- Includes crowns, bridges, dentures, veneers
- Price range: 650-12,740 SEK

**4. Prosthetics (851-852, 841-843)** - Specialized procedures
- Major cost items for patients
- Complete denture: 3,650-4,380 SEK
- Bridge: 3,650-8,760 SEK depending on complexity

## Data Quality Notes

- **Missing values:** Some treatments are only available in general or specialist setting
  - Example: Hygienist exams (111-114) only available in general setting (no specialist price)
  - Example: Some specialist consultations (115-116) only available in specialist setting

- **Material specifications:** Most procedures do not have specific material costs listed
  - Advanced restorative procedures may include specific materials in pricing

- **Price formatting:** All prices are in SEK (Swedish Kronor)
  - Prices represent reference prices, not binding charges
  - Actual patient costs may vary by provider and insurance

## Code System Logic

The Swedish dental code system uses two-digit prefixes:

- **1xx:** Diagnostics (examinations, X-rays, CT scans)
- **2xx:** Prevention and basic treatment
- **3xx:** Root canal and emergency treatment
- **4xx:** Periodontals and extractions
- **5xx:** Surgical root canal treatment
- **6xx:** Bite splints and occlusal treatment
- **7xx:** Fillings and restorations
- **8xx:** Major restorations (crowns, bridges, dentures)
- **9xx:** Implants and orthodontics

## Implementation Files

Two TypeScript files have been generated:

### 1. `dental-reference-prices.ts`
- Complete array of all 200 treatments
- Type-safe interface: `DentalReferencePrice`
- Helper functions:
  - `getCategoryForCode(code: string)` - Get treatment category from code
  - `getPriceRange(codes: string[])` - Get min/max price for multiple codes

### 2. `patient-dental-treatments.ts`
- Simplified patient-facing categories (12 main groups)
- `PatientTreatmentCategory` interface with examples and price ranges
- Helper functions:
  - `findTreatmentByKeyword()` - Search treatments by Swedish or English
  - `getTreatmentsByPriceRange()` - Filter by patient budget

## Usage Examples

### Get price for a specific treatment:
```typescript
import { dentalReferencePrices } from '@/lib/dental-reference-prices';

const rootCanal = dentalReferencePrices.find(p => p.code === '501');
console.log(`Root canal cost: ${rootCanal.generalPrice} SEK`);
```

### Find treatments in patient's budget:
```typescript
import { patientFacingTreatments, getTreatmentsByPriceRange } from '@/lib/patient-dental-treatments';

const affordable = getTreatmentsByPriceRange(0, 2000);
// Returns: Examination, X-rays, Emergency, Cleaning
```

### Search by keyword:
```typescript
import { findTreatmentByKeyword } from '@/lib/patient-dental-treatments';

const crown = findTreatmentByKeyword('tandkrona');
console.log(`Crown price range: ${crown.priceRangeSEK.min}-${crown.priceRangeSEK.max} SEK`);
```
