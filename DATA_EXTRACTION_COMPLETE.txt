IMPLEMENTATION GUIDE
================================================================================

1. IMPORT THE DATA:
   import { dentalReferencePrices } from '@/lib/dental-reference-prices';
   import { patientFacingTreatments } from '@/lib/patient-dental-treatments';
   import { treatmentScenarios } from '@/lib/dental-treatment-quick-reference';

2. FIND A TREATMENT:
   const treatment = dentalReferencePrices.find(t => t.code === '701');
   console.log(`${treatment.description}: ${treatment.generalPrice} SEK`);

3. GET CATEGORY:
   import { getCategoryForCode } from '@/lib/dental-reference-prices';
   const category = getCategoryForCode('701');  // "Ortodonti (Orthodontics)"

4. PRICE RANGE:
   import { getPriceRange } from '@/lib/dental-reference-prices';
   const range = getPriceRange(['701', '702', '703']);
   console.log(`${range.min}-${range.max} SEK`);

5. PATIENT SEARCH:
   const crown = findTreatmentByKeyword('tandkrona');
   console.log(`Crown: ${crown.priceRangeSEK.min}-${crown.priceRangeSEK.max} SEK`);

6. BUDGET FILTER:
   const affordable = getTreatmentsByPriceRange(0, 2000);
   affordable.forEach(t => console.log(`${t.swedishName}: ${t.priceRangeSEK.min}-${t.priceRangeSEK.max} SEK`));

7. SCENARIO EXAMPLE:
   const scenario = treatmentScenarios.find(s => s.id === 'cavity-filling');
   console.log(`${scenario.title}: ${scenario.estimatedCostGeneralSEK.min}-${scenario.estimatedCostGeneralSEK.max} SEK`);
